Thank You For Downloading Xenue Bar


How To Setup Xenue Bar:

1. After installing Xenue Bar, open File Explorer > Downloads >
right click the zip file and click "Extract Here" or "Extract All"

2. Open the new created folder > Xenue Bar > xenuebar.exe

How To Run Xenue Bar On Startup:

1. Right click xenuebar.exe and click pin to start

2. Open start (press windows button) and right click the
xenuebar shortcut. Click on "open file location"

3. Copy the xenue bar shortcut

4. Press "win + r" and type "shell:startup" in the box
Paste the copied shortcut in the opened folder
Xenue Bar will now run on startup on your system
